## 📄 Certificados

Los certificados van en: `/public/certificates/`

```
/public/certificates/
├── tryhackme.pdf (o .jpg)
├── hackthebox.pdf (o .jpg)
├── forense.pdf (o .jpg)
└── ciberseguridad.pdf (o .jpg)
```

**Formato aceptado:** PDF, JPG o PNG

---

## 📐 Formato recomendado para imágenes:
- **Formato:** JPG o PNG
- **Resolución:** 1920x1080 o similar (16:9)
- **Peso:** Optimiza las imágenes (< 500KB por imagen)

## 💡 Tip:
Puedes usar herramientas como TinyPNG, ImageOptim o Squoosh para optimizar el peso sin perder calidad.

---

## 🚀 Cómo agregar tus imágenes:

1. Toma capturas de pantalla de tus proyectos
2. Optimiza las imágenes
3. Copia cada imagen a su carpeta correspondiente
4. Respeta los nombres exactos de los archivos
5. Recarga el navegador (Ctrl+Shift+R) para ver los cambios
